## 授课教师

- 董思聪（方言）
  - 授课风格：比较随和，年轻老师讲课也比较有趣，对论文的批改也很认真负责
  - 听课建议：论文选题围绕方言展开，可以研究家乡方言某个词的起源、一些方言特色等等。（貌似也有人写网络流行语）
  - 先交短文 2000 字，再交长文 5000 字（短文是长文的摘要），并在最后一节课上围绕论文进行 3min 的汇报（老师会掐表）
